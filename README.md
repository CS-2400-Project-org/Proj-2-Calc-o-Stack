# Project additions  
## evaluatePostfix multi digit input  
Evaluate postfix reads integers one digit at a time.  
Ex: 23+ = 2 + 3  
Parenthesis can be used to group digits.  
Ex: (12)+2 = 12 + 3
Parenthesis around single digits will still read as a single digit  
Ex: (2)3+ = 2 + 3
